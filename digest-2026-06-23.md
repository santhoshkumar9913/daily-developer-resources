## 💡 Coding Tip of the Day
**Avoid Agentic Memory Bloat**
While bolting on comprehensive long-term memory tools seems beneficial, recent research shows that excessive unpruned context can actually degrade the reasoning performance of foundational AI models. Instead, prune your context window and use dynamic semantic search to only inject highly relevant, lightweight memory snippets.
